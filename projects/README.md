# Projects

Use this folder for complete builds, experiments, or applications that may involve one or more devices.

A device-specific proof-of-concept can live under `devices/<device>/<tech-stack>/projects/`. A build that combines hardware, wiring, configuration, and a purpose should live here.

Recommended project layout:

```text
projects/<project-name>/
  README.md
  wiring.md
  bill-of-materials.md
  notes.md
  devices/
    <device-folder>/
      <tech-stack>/
        config.yaml
```
